# Tiến độ dự án

## Những gì đã hoạt động

- **Thiết lập dự án:** Cấu trúc thư mục cho cả back-end và front-end đã được tạo.
- **Ngân hàng bộ nhớ:** Đã được khởi tạo với các tệp tài liệu cốt lõi, cung cấp một cái nhìn tổng quan về dự án.

## Những gì cần xây dựng

### Back-end
- [ ] Triển khai xác thực và phân quyền người dùng (đăng ký, đăng nhập).
- [ ] Xây dựng các API endpoint cho quản lý sản phẩm (CRUD).
- [ ] Xây dựng các API endpoint cho quản lý danh mục.
- [ ] Xây dựng các API endpoint cho giỏ hàng và đặt hàng.

### Front-end
- [ ] Xây dựng giao diện cho các trang xác thực (đăng ký, đăng nhập).
- [ ] Xây dựng giao diện trang chủ hiển thị sản phẩm.
- [ ] Xây dựng giao diện trang chi tiết sản phẩm.
- [ ] Xây dựng giao diện giỏ hàng và thanh toán.
- [ ] Xây dựng trang quản trị để quản lý dữ liệu.

## Trạng thái hiện tại

Dự án đang ở giai đoạn đầu của quá trình phát triển. Nền tảng cơ bản đã được thiết lập, và các bước tiếp theo sẽ tập trung vào việc xây dựng các tính năng cốt lõi.
